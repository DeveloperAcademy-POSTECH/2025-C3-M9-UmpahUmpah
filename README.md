## ✅ 테스트 방법

1. 이 저장소를 클론합니다.
```bash
git clone https://github.com/DeveloperAcademy-POSTECH/2025-C3-M9-UmpahUmpah.git
```
2. `Xcode`로 `.xcodeproj` 또는 `.xcworkspace` 열기
3. 시뮬레이터 환경 설정: iPhone 15 / iOS 17
4. `Cmd + R`로 실행 / `Cmd + U`로 테스트 실행
